Challenge 4: Hidden Palindromes

Instructions

A palindrome is a word that is the same forwards and backwards. For example, racecar backwards still spells racecar.

You have been given a list of 500 words in the code block below. Copy the list from the block and paste it into your editor. Write a script that calculates how many words are palindromes in the list.
