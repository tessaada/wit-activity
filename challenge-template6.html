Challenge 6

Instructions

You are writing the logic for a security door that decides who is allowed to enter.

Inputs

  • has_key
  • knows_code
  • is_employee
  • is_visitor
  • is_blocked

Rules

  1. If is_blocked is True → the person cannot enter.
  2. If someone is BOTH an employee AND a visitor → they cannot enter.
  3. Employees can enter if they have a key OR know the code.
  4. Visitors can enter ONLY if they know the code.
  5. Everyone else cannot enter.

Task

Write a function called can_enter() that follows all the rules above.

After writing your function, call it using this exact test case:

can_enter(True, True, True, True, False)

Print the result.

Your program will print either True or False.

Type that exact result into the Final Answer box below.

Important: Do NOT change the test case values.
